    • Gitolite

    A gem which makes configuring your own Git hosting easy ;)

    Project maintained by jbox-web Hosted on GitHub Pages — Theme by mattgraham

    This gem is designed to provide a Ruby interface to the Gitolite Git backend system.

    It provides these functionalities :

    • SSH Public Keys Management
    • Repositories Management
    • Gitolite Admin Repository Bootstrapping

    You can follow announcements here or take a look at the roadmap.

    Code status

    Requirements

    • Ruby 1.9.x or 2.0.x
    • a working gitolite installation
    • the gitolite-admin repository checked out locally

    Installation

    gem install jbox-gitolite

    Read the documentation and more in the Wiki.
